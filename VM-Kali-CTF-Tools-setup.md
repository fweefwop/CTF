## Install Kali Linux Virtual Images     
1. Go to: https://www.offensive-security.com/kali-linux-vm-vmware-virtualbox-image-download/                  
2. Download Kali Linux VMware 64-Bit(7z)                  
3. Unzip into a folder                

## Install CTF Tools
